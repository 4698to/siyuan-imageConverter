# siyuan-imageConverter（中文说明）

[English](./README.md)

`siyuan-imageConverter` 是一个思源笔记插件，用于批量处理当前文档中的图片资源，支持压缩转换和归档整理。

## 功能特性

- 支持一键处理当前文档中的图片
- 支持图片压缩（输出格式可选 `webp` / `avif`）
- 支持图片归档（按笔记本与文档路径分类保存）
- 支持“是否删除原图”开关
- 支持中英文界面文案
- 仅处理本地图片资源，自动跳过网络图片（`http/https/data/file://`）

## 使用方式

1. 打开任意文档，点击右上角插件按钮。
2. 在弹出菜单中选择：
   - `压缩图片`：按当前设置执行压缩/转换流程。
   - `归档图片`：仅归档，不做压缩。

## 设置项说明

- `启用图片压缩`  
  开启后，处理图片时会先压缩再上传；关闭则仅做路径归档/整理。

- `选择格式`  
  压缩输出格式，支持 `webp` 或 `avif`。

- `资源归档`  
  开启后，图片按文档路径归档到 `assets/...` 目录。  
  关闭后，压缩后的图片保存在原图片目录。

- `删除原图`  
  控制转换后是否删除原始图片。  
  当开启“资源归档”时，该选项会被锁定（归档本质为移动资源）。

## 处理范围与注意事项

- 当前版本以“当前编辑文档”为处理范围。
- 菜单操作依赖当前编辑上下文（切换文档或点击编辑区后会刷新上下文）。
- 建议先备份重要资源，再进行批量处理。
